# 📸 사진 라이브러리 권한 설정 가이드

## ✅ Info.plist에 권한 추가하기

Xcode에서 다음 단계를 따라주세요:

### 방법 1: Xcode Target 설정에서 추가 (추천)

1. **Xcode**에서 프로젝트 네비게이터에서 `BlurMate` 프로젝트 클릭
2. **TARGETS** → `BlurMate` 선택
3. **Info** 탭 클릭
4. **Custom iOS Target Properties** 섹션에서 `+` 버튼 클릭
5. 다음 키를 추가:
   - **Key**: `Privacy - Photo Library Additions Usage Description`
   - **Value**: `블러 처리된 사진을 저장하기 위해 사진 라이브러리 접근이 필요합니다.`

### 방법 2: Info.plist 파일 직접 편집

BlurMate 폴더에 `Info.plist` 파일이 있다면:

```xml
<key>NSPhotoLibraryAddUsageDescription</key>
<string>블러 처리된 사진을 저장하기 위해 사진 라이브러리 접근이 필요합니다.</string>
```

---

## 🧪 테스트 방법

1. 앱 실행
2. 사진 불러오기
3. 블러 처리
4. **SAVE** 버튼 클릭
5. 권한 요청 팝업이 뜨면 **허용** 클릭
6. 사진 앱에서 저장된 이미지 확인!
